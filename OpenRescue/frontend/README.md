# OpenRescue Frontend

React + Vite dashboard for incident reporting and live map monitoring.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Frontend runs on `http://localhost:5173` by default.

## Backend URL

Set backend URL with:

```bash
VITE_BACKEND_URL=http://127.0.0.1:8000
```

If not provided, frontend uses `http://127.0.0.1:8000`.

## Build

```bash
npm run build
npm run preview
```
